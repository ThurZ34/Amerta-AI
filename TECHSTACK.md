# Teknologi & Arsitektur yang Digunakan

---

## Laravel 12

<img src="https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white" alt="Laravel Badge" />

Framework PHP untuk backend dan frontend, menyediakan struktur MVC, keamanan, dan fitur modern.

---

## Tailwind CSS

<img src="https://img.shields.io/badge/Tailwind_CSS-06B6D4?style=for-the-badge&logo=tailwindcss&logoColor=white" alt="Tailwind CSS Badge" />

Framework CSS utility-first untuk styling cepat, responsif, dan konsisten tanpa meninggalkan HTML/Blade.

---

## Livewire

<img src="https://img.shields.io/badge/Livewire-EE6983?style=for-the-badge&logo=livewire&logoColor=white" alt="Livewire Badge" />

Membuat komponen UI interaktif dengan PHP (tanpa JavaScript), menggabungkan logika backend dan tampilan frontend secara seamless.

---

## Alpine.js

<img src="https://img.shields.io/badge/Alpine.js-73C7C7?style=for-the-badge&logo=alpinelinux&logoColor=light" alt="Alpine.js Badge" />

Library JavaScript ringan untuk menambahkan interaktivitas UI (modal, dropdown, toggle) secara sederhana dan reaktif.

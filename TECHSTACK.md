# Teknologi & Arsitektur yang Digunakan

## Laravel 12  
Framework PHP untuk backend dan frontend, menyediakan struktur MVC, keamanan, dan fitur modern.

## Tailwind CSS  
Framework CSS utility-first untuk styling cepat, responsif, dan konsisten tanpa meninggalkan HTML/Blade.

## Livewire  
Membuat komponen UI interaktif dengan PHP (tanpa JavaScript), menggabungkan logika backend dan tampilan frontend secara seamless.

## Alpine.js  
Library JavaScript ringan untuk menambahkan interaktivitas UI (modal, dropdown, toggle) secara sederhana dan reaktif.
